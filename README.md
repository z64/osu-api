# Osu::Api

REST API binding for Osu!

See [here](https://github.com/ppy/osu-api/wiki) for more details.

Created for [ccr](https://github.com/z64/ccr)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'osu-api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install osu-api
